Introduzione

Cos'è questa dashboard

La Finance 2026 Dashboard Operativa è un'applicazione web che trasforma i file Excel di pianificazione in una console interattiva per il controllo di gestione.

Funziona interamente nel browser: non richiede installazioni, server o connessione internet. Basta aprire il file HTML e caricare i propri dati Excel per avere immediatamente accesso a grafici, KPI e tabelle dettagliate.

💡
Come funziona I dati vengono elaborati localmente nel browser tramite la libreria XLSX.js. Nessun dato viene inviato a server esterni. La dashboard è un file .html standalone che puoi salvare e usare offline.

File richiesti § 2

Requisiti e formato file

La dashboard accetta file in formato .xlsx o .xls. È necessario caricare almeno il file principale di pianificazione; il file Budget Commerciale è opzionale e sblocca funzionalità aggiuntive.

File Nome atteso Fogli necessari Stato
Pianificazione operativa Pianificazione_operativa_FINANCE_2026_*.xlsx dettaglio, allocazione, commesse, pivot_costi Obbligatorio
Budget Commerciale Monitoraggio_commerciale_iniziative_*.xlsx Foglio con colonna Codice commessa Opzionale
⚠️
Attenzione al nome del file La dashboard riconosce i file in base al nome. Se il file di pianificazione non contiene la parola Pianificazione o FINANCE nel nome, potrebbe non essere riconosciuto correttamente. Il sistema mostra comunque un messaggio di errore esplicativo.

Il foglio principale deve contenere una riga di intestazione con almeno le seguenti colonne (la ricerca è case-insensitive):

Caricamento § 3

Schermata di caricamento

All'apertura del file HTML viene mostrata una schermata di caricamento. È il punto di partenza per importare i propri dati nella dashboard.

  1. 1
    Trascina i file nell'area tratteggiata oppure clicca su di essa per aprire il selettore file. Puoi selezionare più file contemporaneamente tenendo premuto Ctrl / ⌘.
  2. 2
    Controlla l'elenco dei file rilevati: accanto ad ogni file apparirà un'etichetta che indica se è stato riconosciuto come "Pianificazione" o "Budget comm.". I file non riconosciuti vengono ignorati.
  3. 3
    Seleziona l'anno di riferimento (es. 2025, 2026) tramite i pulsanti che appaiono sopra la drop zone se il file contiene più anni.
  4. 4
    Clicca "Avvia Dashboard". La dashboard elabora i dati (operazione di pochi secondi) e mostra una barra di avanzamento con la fase corrente.
✅
Caricamento separato del Budget Se in un secondo momento vuoi aggiungere il file Budget Commerciale, puoi farlo senza ricaricare tutto: nella barra in alto trovi il pulsante + Budget comm. che permette di aggiungerlo a dashboard già aperta.
🔄
Cambiare file Per caricare un file diverso, usa il pulsante ↺ Cambia file in alto a destra. Questo riporta alla schermata iniziale e cancella i dati correnti dalla memoria del browser.
Interfaccia § 4

Struttura e navigazione

Una volta caricati i dati, la dashboard è organizzata in tre aree principali:

🔝
Barra superiore (Header)
Informazioni sul file caricato e selettore anno
▼
Mostra il nome del file caricato, la versione della dashboard e — se il file contiene più anni — il selettore anno (pulsanti 2025 / 2026 ecc.). Cambiare anno ricalcola immediatamente tutte le metriche. In questa barra trovi anche il pulsante + Budget comm. e il pulsante ↺ Cambia file.
🗂️
Barra di navigazione (Tab bar)
Le 9 sezioni analitiche accessibili con un click
▼
Subito sotto l'header si trova la barra con le 9 tab numerate da ① a ⑨. Cliccando su una tab si carica la sezione corrispondente. La tab attiva è evidenziata in blu con una linea sotto. Su schermi stretti la barra è scrollabile orizzontalmente.
📊
Area contenuto principale
Grafici, KPI e tabelle per la sezione attiva
▼
L'area principale occupa il resto dello schermo e cambia dinamicamente in base alla tab selezionata. Molte sezioni hanno una striscia KPI in cima con i valori chiave, seguita da grafici e tabelle. Le tabelle sono spesso cliccabili: selezionando una riga si apre un pannello di dettaglio laterale o sottostante.

Striscia Ricavi (se presenti)

Se il file contiene dati di ricavo, sotto la striscia KPI appare una sezione dedicata con:

Tab ② § 6

② Andamento Mensile

Analisi dettagliata dell'evoluzione mese su mese di costi, giornate e composizione per tipo di attività.

La sezione contiene tre grafici affiancati:

Grafico Combo (barre + linea): le barre mostrano i costi mensili in euro, la linea sovrapposta mostra le giornate. Permette di verificare la correlazione tra volume di lavoro e spesa.
Bar chart per tipo attività: ogni barra è suddivisa per tipologia (es. Produzione, Sviluppo, Management…). Utile per capire come cambia il mix di attività nel corso dell'anno.
Working Days chart: confronto tra giornate allocate e giornate lavorative teoriche per mese. Evidenzia mesi di sovra- o sotto-allocazione.
Tab ③ § 7

③ Unit & Profili

Breakdown della spesa e delle giornate per unità organizzativa e per profilo/tipologia di risorsa.

Quattro grafici visualizzano le distribuzioni:

  • Costi per Unità — bar chart orizzontale ordinato per valore decrescente
  • Top Commesse — le commesse con maggiore incidenza sui costi
  • Distribuzione Profili — mix dei profili professionali presenti nel piano
  • Tipo Risorsa — distribuzione tra risorse interne, esterne, ecc.

📊

Tab ⑦ § 11

⑦ Analisi Commesse

Approfondimento interattivo su ogni singola commessa: dall'andamento mensile dei costi al dettaglio delle risorse allocate.

La sezione ha un layout a pannelli affiancati:

  1. A
    Lista commesse a sinistra con KPI aggregati in cima (totale pianificato, mesi attivi, ecc.). Usa ↑ / ↓ da tastiera per navigare la lista.
  2. B
    Pannello dettaglio a destra: si attiva cliccando su una commessa. Mostra un mini-chart con i costi mensili della commessa e una tabella con tutte le allocazioni risorsa/mese.
⌨️
Navigazione da tastiera Con la lista commesse in focus, puoi usare i tasti freccia ↑ e ↓ per scorrere rapidamente da una commessa all'altra senza usare il mouse. Il pannello di dettaglio si aggiorna in tempo reale.

Strumento § 14

Budget Commerciale (opzionale)

Il file Budget Commerciale è un secondo file Excel che, se caricato, sblocca funzionalità aggiuntive di confronto tra pipeline commerciale e pianificazione operativa.

Come caricarlo

Puoi caricarlo in due modi:

  • All'avvio — trascina entrambi i file nell'area di drop durante la schermata di caricamento iniziale
  • A dashboard aperta — clicca il pulsante + Budget comm. nell'header per aggiungere il file senza perdere i dati già caricati

Funzionalità sbloccate

📊
KPI Ricavi Budget + WIP
In Tab ① Panoramica
▼
Appaiono due nuovi KPI nella striscia ricavi: Ricavi budget commerciale (commesse con codice) e Ricavi budget comm. in lavorazione (commesse WIP senza codice ancora assegnato). Entrambi sono cliccabili per il drill-down.
🔍
Gap Check commesse mancanti
In Tab ① Panoramica
▼
Il pulsante "Verifica Gap Commesse" confronta le commesse del budget commerciale con quelle nella pianificazione e mostra quelle presenti nel budget ma non ancora pianificate. Utile per tenere allineata la pipeline commerciale con l'esecutivo.
⚖️
Sezione ⑨ Allineamento Costi
Dati di confronto arricchiti
▼
La tab ⑨ beneficia dei dati di budget commerciale mensili per arricchire il calcolo del residuo e dell'allineamento per commessa.
Strumento § 15

Export dati (.xlsx)

Ogni sezione con una tabella permette di esportare i dati visualizzati in un file Excel scaricabile direttamente dal browser.

Il pulsante di export (icona ↓ con etichetta xlsx) è posizionato nell'intestazione di ogni tabella esportabile. L'export include:

  • Tutte le righe visibili nella tabella al momento dell'export (inclusi eventuali filtri attivi)
  • Intestazioni di colonna come prima riga
  • Larghezze colonna ottimizzate automaticamente
  • Nome del foglio basato sulla sezione (es. Commesse, Marginalità…)

Sezioni con export disponibile:

④ Commesse ⑤ Allocazione risorse ⑤ Dettaglio risorsa ⑥ Marginalità ⑦ Analisi Commesse ⑧ Analisi Allocazioni ⑧ Risorse Unità ⑨ Allineamento Budget
Risoluzione problemi § 16

Messaggi di errore comuni

Ecco i messaggi che puoi incontrare e come risolverli.

🔴
"Formato non supportato"
Il file caricato non è un .xlsx o .xls
▼
La dashboard accetta solo file Excel in formato .xlsx o .xls. Assicurati di non caricare file .csv, .ods o altri formati. Se hai un CSV, aprilo in Excel e salvalo come .xlsx.
🔴
"Il foglio dettaglio non è stato trovato"
Struttura del file non riconosciuta
▼
Il file Excel caricato non contiene un foglio chiamato dettaglio (ricerca case-insensitive). Verifica di star caricando il file corretto: Pianificazione_operativa_FINANCE_2026_*.xlsx. Se il foglio ha un nome diverso, contatta chi gestisce il file sorgente.
🟡
"Il file non contiene dati riconoscibili"
File vuoto o struttura diversa dall'atteso
▼
Il file viene riconosciuto come Excel ma il suo contenuto non corrisponde alla struttura attesa. Verifica che il file non sia vuoto e che le colonne abbiano i nomi corretti (vedi §2 — Struttura del foglio). Per maggiori dettagli tecnici, apri la console del browser con F12 e controlla i messaggi di log.
🟡
Budget Commerciale caricato ma Pianificazione assente
Solo il file opzionale è stato caricato
▼
Il file Budget Commerciale è stato rilevato, ma manca il file principale di Pianificazione. La dashboard mostra un avviso prominente e non può procedere. Assicurati di caricare anche Pianificazione_operativa_FINANCE_*.xlsx.
🔵
Sezione ⑨ mostra dati parziali o vuoti
Foglio pivot_costi non disponibile
▼
La sezione Allineamento Costi richiede il foglio pivot_costi nel file di pianificazione. In alcune versioni del file questo foglio potrebbe non essere presente o potrebbe essere un foglio pivot che XLSX.js non riesce a leggere. In questo caso la sezione mostrerà solo i dati disponibili senza il confronto M10.
🛠️
Debug avanzato Per qualsiasi errore non previsto, apri la console del browser (F12 → tab "Console") e controlla i messaggi di warning o errore. La dashboard registra informazioni dettagliate sulle operazioni di lettura file che possono aiutare a diagnosticare problemi di struttura.
